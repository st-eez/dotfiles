# Technology Stack - Steez Dotfiles

## Operating Systems
- **macOS** (Primary target)
- **Arch Linux** (Secondary support)
- **Ubuntu / Linux Mint** (Debian-based support)

## Core Tools
- **Shell**: Zsh (Framework: Oh-My-Zsh, Theme: Powerlevel10k)
- **Terminal Emulator**: Ghostty
- **Editor**: Neovim (Distribution: LazyVim)
- **Window Management**:
    - macOS: Aerospace (Tiling), Borders
    - Menu Bar: Sketchybar

## Management & Automation
- **Dotfile Manager**: GNU Stow
- **Installer UI**: Charm Gum
- **Package Managers**:
    - macOS: Homebrew
    - Arch: Pacman / AUR (Paru/Yay)
    - Debian/Ubuntu: APT

## Utilities
- **Navigation**: zoxide, fzf
- **Search**: ripgrep (rg), fd
- **File Listing**: eza
- **Monitoring**: btop
- **Git**: lazygit, gh (GitHub CLI)
- **Secrets**: Bitwarden
- **Launcher**: Raycast (macOS)
